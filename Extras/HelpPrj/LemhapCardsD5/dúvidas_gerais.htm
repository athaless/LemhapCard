n O que é Plug-In?
São módulos de programa adicionais que podem ser instalados (acoplados) ao programa original a critério do usuário. Plug-ins permitem aumentar a funcionalidade do programa conforme as necessidades do usuário. Cada fabricante de software tem suas normas para criação de Plug-Ins e os Plug-ins compatíveis com Cartões Lemhap estarão disponíveis a medida que forem criados para atender novas funcionalidades do usuário.

n Qual o significado do número de versão do sistema?
O número de versão do sistema segue o padrão indicado pela Borland:"Version","Minor Version","Release","Build".
Version: Versão do sistema em si, determina o número da versão atual do sistema, este número sendo diferente entre versões indica uma grande mudança estrutural e lógica entre os mesmos.
Minor Version: Indica pequenas alterações sobre uma versão, adição ou retirada de novas funcionalidades. Ou um conjunto de dez releases.
Release: Indica alterações relativas a correções de Bugs sobre uma "Minor Version", pode também indicar modificações internas (sem efeito aparente ao usuário final). A cada dez adiciona-se uma Minor Version e zeramos o release.
Build: Número de vezes que o programa fora compilado desde sua criação. Controle para os desenvolvedores.
