Criando um trabalho


Para a criação de um trabalho pode-se especificar diretamente as propriedades ou escolher um modelo para usar como referência. Os dois botões indicados abaixo permitem que o usuário escolha uma ou outra opção, respectivamente.


Definindo diretamente as propriedades
Se for escolhida a primeira opção aparecerá uma janela como a indicada na imagem abaixo a esquerda com campos a serem definidos pelo usuário. O próprio usuário estará livre para definir as dimensões do seu trabalho ou poder escolher um tipo entre os tipos já cadastrados na aplicação para cartões, envelopes, folhas A4 e etc.

Escolhendo um modelo
Para a segunda opção aparecerá uma janela como a exibida abaixo onde o usuário poderá selecionar um dos modelos de cartão, envelope, papel timbrado ou outros para criar seu trabalho.
